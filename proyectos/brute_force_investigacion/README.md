# Brute Force Investigation

## Objetivo 
Anaizar un posible  ataque  de fuerza bruta utilizando registros de autenticacion

## Herramientas utilizadas
 - CSV
 - MITRE ATT&CK
 - GitHub

 ## Actividades 
  - Analisis de logs
  - Identificacion de actividad sospechosa
  - Relacion con MITRE ATT&CK
  - Elaboracion  de un informe de incidente

## Habilidades Desarrolladas
 - Analisis de eventos 
 - Investigacion inicial de incidentes 
 - Pensamiento critico
 - Docuentacion tecnica
