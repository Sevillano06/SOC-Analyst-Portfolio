# 🛡️ Blue Team & SOC Analyst Portfolio

¡Bienvenido a mi repositorio de ciberseguridad! Este espacio está diseñado para documentar mi trayectoria técnica, laboratorios prácticos y metodologías de investigación enfocadas en Operaciones de Seguridad (SOC), Análisis de Defensa y Detección de Amenazas.

El objetivo de este portafolio es demostrar habilidades sólidas en la gestión de logs, análisis de alertas, investigación de incidentes y uso de herramientas SIEM a nivel corporativo.
