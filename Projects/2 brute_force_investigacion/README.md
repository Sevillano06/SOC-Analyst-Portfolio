# 🚨 Proyecto 02: Brute Force Investigation

## 🎯 Objetivo 
Analizar un posible ataque de fuerza bruta utilizando registros de autenticación.

## 🛠️ Herramientas utilizadas
- Análisis de archivos **CSV**
- Framework **MITRE ATT&CK** (Mapeo de tácticas y técnicas)
- **GitHub** (Control de versiones y documentación)

## 📋 Actividades realizadas
- Análisis detallado de logs de seguridad.
- Identificación de patrones de actividad sospechosa (intentos de inicio de sesión).
- Relación del comportamiento anómalo con el framework MITRE ATT&CK (ej. *Tactic: Credential Access, Technique: T1110 - Brute Force*).
- Elaboración de un informe de incidente estructurado.

## 🧠 Habilidades Desarrolladas
- Análisis de eventos de seguridad (Event Analysis).
- Investigación inicial de incidentes (Triaje).
- Pensamiento crítico aplicado a la defensa cibernética.
- Documentación técnica y reporte.
