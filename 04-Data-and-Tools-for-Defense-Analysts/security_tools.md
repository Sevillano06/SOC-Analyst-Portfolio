# Herramientas del Analista SOC

| Herramienta | Función | Ejemplo |
| :--- | :--- | :--- |
| **SIEM** | Centraliza, normaliza y analiza eventos y logs. | Splunk, Microsoft Sentinel |
| **EDR** | Monitorea actividad de procesos y red en endpoints. | Defender for Endpoint, CrowdStrike |
| **IDS** | Detecta y alerta sobre intrusiones en la red. | Snort, Zeek |
| **IPS** | Detecta y bloquea activamente ataques en curso. | Suricata |
| **Firewall** | Filtra tráfico de red entrante/saliente. | Fortinet, Palo Alto |
| **Threat Intel** | Proporciona indicadores de compromiso (IoCs). | MISP, VirusTotal |
