# 🏢 Operaciones de Seguridad y el Analista de Defensa

Este módulo está destinado al estudio profundo de las operaciones dentro de un entorno SOC, abarcando las metodologías de triaje, la gestión de incidentes bajo estándares internacionales y las funciones críticas del Analista de Defensa (Defensive Analyst).

---

## 🎯 Objetivos de Aprendizaje Próximos
* **Comprender la Arquitectura del SOC:** Analizar cómo interactúan el factor humano, los procesos de negocio y las tecnologías de automatización.
* **Dominar el Ciclo de Vida de Incidentes:** Estudiar los frameworks de manejo de incidentes basados en **NIST (SP 800-61)** y **SANS**.
* **Modelado de Playbooks:** Analizar la automatización de respuestas ante incidentes comunes como Phishing, Malware y accesos no autorizados.
